# Alcoste Tattoo 🖋️

Portfolio web minimalista para el estudio de tatuajes **Alcoste Tattoo** en Cáceres. Enfoque en velocidad, diseño responsive y SEO local.

## 🚀 Stack
- **Framework:** React + Vite
- **Estilos:** Tailwind CSS
- **Componentes:** Shadcn UI
- **Navegación:** React Router
- **Hosting:** Vercel
